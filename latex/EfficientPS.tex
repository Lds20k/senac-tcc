\subsection{EfficientPS}

EfficientPS é uma solução para a segmentação panóptica proposta no artigo \citeonline{mohan2020efficientps}, o trabalho apresenta uma arquitetura nova baseada em decodificação multicamadas seguido  de uma bifurcação para uma arquitetura de segmentação semântica e de instância e no fim uma arquitetura para combinar os resultados, este processo é ilustrado na \cref{fig:arqEP}.
